**InklyAI**

InklyAI is an intelligent writing companion that helps you improve grammar, fix spelling mistakes, and rephrase text using AI.

Features:

✅ AI Grammar Correction – Detects and fixes grammatical errors using advanced language models.

✅ Smart Spell Checker – Context-aware spelling correction for clean, professional writing.

✅ Intelligent Rephrasing – Rewrites sentences for clarity, tone, and impact.

✅ Secure Login with Privy – Web3-friendly authentication powered by Privy
.

✅ Modern UI – Built with React and Tailwind CSS for a smooth, responsive design.
